# NGStats — Newgrounds Audio Statistics Overlay

Плавающее Android-приложение для парсинга и анализа статистики аудио-работ на Newgrounds.com

---

## Возможности

- 🪟 **Плавающее окно** поверх любых приложений (Android Overlay)
- 🔍 **Парсинг профиля** по ссылке или username
- 📊 **Таблица с сортировкой** по названию, просмотрам, рейтингу, комментариям, году
- 📅 **Фильтр по году** — отдельно за 2022, 2023, 2024 или за всё время
- ☑️ **Настройка колонок** — включай/выключай: Views, Score, Comments, Year, Genre
- 📈 **Сводная статистика** — всего треков, суммарные просмотры, средний рейтинг, топ трек

---

## Сборка через Android Studio (рекомендуется)

1. Установите [Android Studio](https://developer.android.com/studio)
2. Откройте папку `NGStats` как проект (`File → Open`)
3. Дождитесь синхронизации Gradle (скачает зависимости ~50 MB)
4. Нажмите **Build → Build Bundle(s) / APK(s) → Build APK(s)**
5. APK появится в `app/build/outputs/apk/debug/app-debug.apk`

---

## Сборка через командную строку

```bash
# Убедитесь, что ANDROID_HOME и JAVA_HOME настроены
cd NGStats
chmod +x gradlew        # Linux/Mac
./gradlew assembleDebug # Linux/Mac
# или
gradlew.bat assembleDebug # Windows
```

APK будет в `app/build/outputs/apk/debug/app-debug.apk`

---

## Установка на телефон

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

Или скопируйте APK на телефон и откройте файловым менеджером (нужно разрешить установку из неизвестных источников).

---

## Первый запуск

1. Откройте **NGStats**
2. Нажмите **«Запустить окно статистики»** — откроется системный экран разрешений
3. Включите **«Разрешить отображение поверх других приложений»** для NGStats
4. Вернитесь — приложение автоматически запустит плавающее окно

## Использование оверлея

```
┌─────────────────────────────────────┐
│ 🎵 NGStats                      [▲] │  ← потяни для перемещения
├─────────────────────────────────────┤
│ [malego16.newgrounds.com/audio] [Парсить] │
│ [Все годы ▾]                [Фильтр] │
│ ☑Views ☑Score ☑Comm ☑Year ☐Genre   │
├─────────────────────────────────────┤
│ Треков: 65 | Просмотров: 18,234     │
│ Avg ★: 3.45 | Топ: HELP! (3119)    │
├─────────────────────────────────────┤
│ Название ↕  │Views↕│★↕  │Comm↕│Year↕│
│ HELP!       │ 3119 │4.20│  12 │2023 │
│ Unknown place│4850 │3.90│   8 │2023 │
│ brEAkgOrE   │ 1697 │4.10│   5 │2023 │
│ ...                                 │
└─────────────────────────────────────┘
```

- **[▲/▼]** — свернуть/развернуть окно
- **Тянуть за заголовок** — перемещать окно
- **Нажать на заголовок колонки** — сортировать (повторное нажатие — обратная сортировка)
- Ввести `malego16` или `https://malego16.newgrounds.com/audio` — оба формата работают

## Важные замечания

- Приложение **не использует официальный API** Newgrounds — парсит HTML страниц.  
  Если Newgrounds изменит разметку, возможно потребуется обновить `NgParser.java`.
- Данные о **рейтинге и комментариях** доступны на странице только если они отображаются публично.
- Сайт Newgrounds может **ограничить запросы** при слишком частом парсинге.

---

## Если что-то не работает

**Оверлей не отображается:**  
→ Настройки → Приложения → NGStats → Дополнительные разрешения → Отображение поверх других

**Ошибка при парсинге:**  
→ Проверьте интернет-соединение; убедитесь что username написан правильно

**Таблица пустая хотя загрузка прошла:**  
→ Возможно Newgrounds изменил HTML-разметку. Откройте `NgParser.java` и обновите CSS-селекторы согласно актуальной разметке страницы.
